Бот, который может записывать данные о продажах и выводить статистику

База данных бота содержит 1000 сгенерированных записей о продажах различных товаров.
Вы можете более детально изучить способ генерации данных в файле datagenerator.

Бот обладает двумя основными функциями, доступными по нажатию одной из кнопок:
1) Добавление записей
    Для добавления записи о продаже необходимо одним сообщением ввести данные,
    согласно указанному формату. Пример сообщения, соответсвующего формату:

2022-01-01
Сахар
50
10

2) Просмотр отчета за период
    Для просмотра отчета нажмите соответсвующую кнопку и укажите интересующий
    Вас интервал дат. Пример сообщения, соответсвующего формату:

2022-01-01
2022-06-25

Обратите внимание:
модуль генерации записей создал даные только за 2022 и 2023 год.
